Privacy Policy

Last updated: May 5, 2026

YourWorld is a code-based content sharing platform. Users can create content and share it with selected people using a unique code. Only people with the code can access shared content.

What data we store

We store the content you create, such as diary pages, story pages, titles, optional image prompts, optional uploaded media links, theme settings, generated share codes, and basic metadata such as creation and update timestamps.

Where data is stored

Content and optional metadata are stored in the application database and cloud storage services used by YourWorld. Local browser storage may also be used for preferences such as theme or audio settings, especially for guest use.

Code-based access

When you generate a share code, that code becomes the access key for the selected content. Anyone with the code can open the shared content. Anyone without the code cannot access it through the sharing page.

Personal data

No personal data is collected or shared without your control. If you sign in, we may store your email, display name, profile picture, and login metadata so you can manage your saved content.

No sale or sharing of user data

We do NOT sell or share user data. We do not sell your content, share codes, email address, or profile data to advertisers or data brokers.

What we do not ask for

YourWorld does not ask visitors opening shared content for banking information, government ID numbers, card details, or unrelated passwords.

Your control

You control what you create and what you share. You can remove or replace share codes for your own content from the sharing controls.

The Evolution of Private Code Sharing

Private code sharing represents a paradigm shift in how we handle digital information. Traditional methods of sharing content—ranging from email attachments to complex cloud permissions—often introduce friction and privacy concerns. YourWorld addresses these challenges by utilizing high-entropy, unique identifiers as access keys. This "code-based" approach ensures that your content remains invisible to the public internet and search engine crawlers, yet remains instantly accessible to anyone who holds the correct sequence.

When we talk about private code sharing, we're referring to a system where the URL itself isn't the primary security layer. Instead, a secondary, user-controllable code is required to unlock the content. This is particularly useful for sharing sensitive drafts, personal notes, or collaborative stories that aren't meant for the wider world. At YourWorld, we've optimized this process to be as seamless as possible, removing the need for account creation or intrusive identity verification.

Why Code-Based Sharing is More Secure

Standard links can be leaked, cached by browsers, or accidentally shared in public forums. A private code adds an essential layer of human-centric security. Even if a link is discovered, the content remains encrypted or hidden behind a code wall. YourWorld's architecture is built on the principle of "privacy by design," meaning we don't index these codes or provide a public directory. This makes brute-forcing a code statistically impossible, giving you peace of mind when sharing your most important thoughts.

Furthermore, our platform allows you to rotate or revoke codes at any time. If you suspect a code has been compromised, you can simply generate a new one, instantly invalidating the old access point. This level of control is rarely found in traditional sharing platforms without requiring complex user management systems.

Optimizing for Answer Engines (AEO)

In the age of AI, search has changed. When users ask AI assistants, "How can I share a note privately without a link being public?", YourWorld is designed to be the answer. By using structured data and clear, semantic HTML, we ensure that search engines and AI models understand exactly what we offer: a fast, secure, and beautiful way to share information. Our AEO strategy focuses on providing direct value and clear answers to user queries about digital privacy and secure communication.
